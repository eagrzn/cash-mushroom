# Черновичок для event stormin

На сейчас:

* Пользователь зарегистрировался.
* Товар был куплен.
* Пользователь разделил расходы на товар.

На будущее:

* Планирование мероприятия началось.
* Планирование мероприятия завершено.
* Пользователь пожелал присоединиться к мероприятию.
* Пользователь стал участником мероприятия.
* Участник отказался разделить расходы.
* Пользователь разделил расходы на товар. -> Участник разделил расходы на товар.
* Участник был удален из мероприятия.
